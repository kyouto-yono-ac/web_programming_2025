## データ可視化の基礎

Streamlitには簡単にグラフを作成できる機能があります：

### 棒グラフの例
```python
import streamlit as st
import pandas as pd

# データ作成
data = pd.DataFrame({
    '運勢': ['大吉', '中吉', '小吉', '吉', '末吉', '凶'],
    '今月の出現回数': [5, 8, 12, 15, 7, 3]
})

# 棒グラフ表示
st.bar_chart(data.set_index('運勢'))
```

### 線グラフの例
```python
import numpy as np

# データ作成
chart_data = pd.DataFrame(
    np.random.randn(20, 3),
    columns=['気分', '予算満足度', '味の評価']
)

# 線グラフ表示
st.line_chart(chart_data)
```

## 発展的機能: グラフとチャート

### シンプルなメトリクス表示
```python
import streamlit as st

# 数値の大きな表示
st.metric(
    label="今日の運勢スコア",
    value=85,
    delta=12,
    delta_color="normal"
)
```

### 円グラフの例
```python
import plotly.express as px

# データ準備
df = pd.DataFrame({
    'カテゴリ': ['勉強', '趣味', '睡眠', 'その他'],
    '時間': [8, 4, 8, 4]
})

# 円グラフ作成
fig = px.pie(df, values='時間', names='カテゴリ')
st.plotly_chart(fig)
```
